# Demo 02: `find` and `grep`

This is a small research project with data, scripts, notebooks, and results.
Your job is to use `find` to navigate and explore it.

## Finding with `find`

- **Task**: there's a "data dictionary" somewhere in this project describing the data. Find it.
- **Task**: how many JSON files are in this project?
- **Task**: are there any image files in this project?
- **Task**: make a backup copy of all of the python files. Copy them to a new `backup/` directory
- **Task**: change all of the extensions of the backed-up files to `.py.bak`. That is, `foo.py` gets renamed as `foo.py.bak`.
- **Bonus Task**: change all of the extensions of the backed-up files to `.bak.py`. That is, `foo.py` gets renamed as `foo.bak.py`.
