# Demo 02: `find` and `grep`

This is a small research project with data, scripts, notebooks, and results.
Your job is to use `find` to navigate and explore it.

## Finding with `find`

- **Task**: there's a "data dictionary" somewhere in this project describing the data. Find it.
- **Task**: how many JSON files are in this project?
- **Task**: are there any image files in this project?
- **Task**: make a backup copy of all of the JSON files. Copy them to a new `backup/` directory
- **Bonus Task**: make that same backup, change all of the extensions to `.json.bak`. That is, `foo.json` gets backed up as `backup/foo.json.bak`.
    

## Finding with `grep`

- **Task**: there are some "fatal" errors in the log file. Find all three.
- **Task**: the experiment has a "test_size" parameter
