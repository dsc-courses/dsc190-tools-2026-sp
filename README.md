# DSC 190: Tools of the Trade - Spring 2026

Welcome to DSC 190: Tools of the Trade. This course will introduce you to the
most important tools for getting stuff done in data science. We'll cover the
command line, version control, managing your development environment, deploying
data science models to production, and more. All of this will be done using the
meta-tool of the future: *agentic AI*.

This repository is also the website. It contains all of the course materials. The
important files and directories are:

- `SYLLABUS.md`: the course syllabus, including the schedule and grading
- `lectures/`: slides and materials used for each lecture.
    - Each lecture is a markdown file (`slides.md`) that is meant to be
      displayed using [presenterm](https://mfontanini.github.io/presenterm/).
    - PDF versions of the slides are linked from [`lectures/README.md`](lectures/README.md).
- `assignments/`: starter code and instructions for each assignment
- `resources/`: supplementary notes and guides on various topics.

We will also use [Gradescope](https://www.gradescope.com/courses/1287049) for
grades and [Campuswire](https://campuswire.com/c/G4177828B/feed) as a messageboard.
You should have received invitations to both.

If you'd like to view this repository on the web, you can find it at:

> https://github.com/dsc-courses/dsc190-tools-2026-sp

This repo is designed to be used by agentic AIs, like Claude Code. There is a full
collection of skills that AIs can use in the `.agents/skills/` directory.
